* Create a new folder and add a package.json file,
this file will be used to install project dependencies.
* Create index.js file in root of directory
* Install node.js from command line 
* Run npm init to initialize package.json file,
proceed to provide the default information.
* Run the below commands to install some necessary dependencies
- npm install jest --save
- npm install mocha --save
- npm install selenium-webdriver --save
- npm install fs --save
- npm install chromedriver --save
- npm install geckodriver --save
